PROJECT CONCEPT OVERVIEW
This project is a modern OpenGL 3D simulation of the Solar System.
It combines rendering, classic graphics algorithms, camera navigation, lighting, texturing, and transformation demos.
Main flow: main loop -> update simulation with delta time -> render skybox/sun/planets/orbits/helpers.
Classic algorithms are integrated as practical visualization tools:
- Midpoint Circle for orbit path generation.
- DDA for axis point generation.
- Bresenham for grid line generation.
Transformations demonstrated in project logic: translation, rotation, scaling, reflection, and shear.
